This game was submitted to Brackeys2025.2 Game Jam, with the theme 
"Risk it for the biscuit." Inspired by the roulette mechanic in the Carnival Mode of 
"For The King 2," it's a wave-based shooting game where a roulette is spun before 
each wave to apply buffs or debuffs to either the enemies or the player.

In line with the theme of risking something to gain a reward, 
the game features a mechanic where the fewer beneficial blocks the player inserts, 
the stronger their effects become.

The team consisted of one programmer, two sound/music designers, and one artist. 
Out of 2,299 entries, the game ranked 255th in Gameplay, its highest score category. 
Overall, it achieved a solid 3.5/5 rating, placing it in the **top 11%** of submissions.
<br>

## Information

- **Engine**: Unity

- **Platform**: WebGL

- **Team Size**: 4 (1 programmer, 1 game designer, 2 sound/music designers)

- **Time Frame**: 7 days
